# Prakse_lietoti_kodi

Šeit ir atrodmas Jupiter Notebooks, kurās var redzēt prakses atskaitē atrodamos rezultātus. Katram no uzdevumiem ir izveidots savs fails.

Failā ```perform_hypothesis_testing.ipynb``` var atrast kodu, kas veic aprakstīto hipotēžu pārbaudi.
Failā ```perform_logistic_regression.ipynb``` var atrast kodu, kas veic loģistiskās regresijas izveidi, lai prognozētu, kuri klienti veiks atkārtotu pirkumu.


Katrā no failiem ir izveidota funkcija, kurai var padot datus un kura veiks atbilstošo uzdevumu. Loģistiskās regresijas gadījumā var padot papildus parametrus, kā proporciju, starp Reliable negatvies un marķētajiem datiem, max IV vērtību, pie kuras prediktors tiek ignorēts un gludināšanas konstanti, kura tiek lietota WoE aprēķiniem. kodi netiek pievienoti pa taisno prakses atskaitē, jo tie aizņemtu pārāk daudz vietas un būtu problēmas tos pievienot kā pielikumus.
