# Prakse_lietoti_kodi

Šeit ir atrodmas Jupiter Notebooks, kurās var redzēt prakses atskaitē atrodamos rezultātus. Katram no uzdevumiem ir izveidots savs fails.

Failā ```perform_hypothesis_testing.ipynb``` var atrast kodu, kas veic aprakstīto hipotēžu pārbaudi.
Failā ```perform_logistic_regression.ipynb``` var atrast kodu, kas veic loģistiskās regresijas izveidi, lai prognozētu, kuri klienti veiks atkārtotu pirkumu.
